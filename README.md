# qq-laliga-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态 HTML 页面的存放与索引使用。

## 项目简介

本仓库的主要目的是收集、整理并托管一批独立的 HTML 文件。这些页面各自包含完整的内容与样式，彼此之间无依赖关系。仓库本身不提供任何后台服务或动态功能，所有页面均为纯前端静态页面。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 可选的主索引页（如存在）
├── pages/             # 存放归档的 HTML 页面（可选）
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/            # 公共资源（如 CSS、JS、图片等，可选）
```

- `pages/` 目录用于存放独立的 HTML 页面文件，每个文件对应一个归档内容。
- `assets/` 目录用于存放页面可能引用的公共资源，如样式表、脚本或图片。
- 根目录下的 `index.html` 可作为仓库的入口页面，展示所有归档页面的列表或说明（如有需要）。

## 页面归档说明

- 每个 HTML 文件均为独立的完整页面，包含其自身的结构与样式。
- 页面之间互不影响，引用资源时请使用相对路径或内联方式，以确保在本地或直接打开时正常工作。
- 归档的页面内容可能来自不同来源或时期，但均以原始形式保存，不做修改或适配。

## 维护说明

- 本仓库由维护者不定期更新，新增或清理归档页面。
- 如需添加页面，请将 HTML 文件放入 `pages/` 目录，并确保文件命名清晰、无冲突。
- 不建议在页面中包含外部链接或依赖外部服务，以保持归档的独立性和可访问性。
- 如有问题或建议，欢迎通过 Issues 反馈。

## 许可

本仓库中的内容仅供归档与参考使用。具体页面的版权归属其原始作者或来源。
